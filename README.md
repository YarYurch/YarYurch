# Гипер-игра


## Описание

Это консольная игра на языке Python, где вы выбираете, кем вам быть, и сражаетесь с врагами в невероятном текстовом формате!

## Функционал

- Выбор класса
- Характеристики: здоровье, мана, оружие, особые атаки, параметры врагов
